1. Introduction

edtFTPnet/Free is one of the most popular .NET FTP client libraries available today. Robust, fast and with an extensive API, it provides a solid foundation for developers who require FTP functionality in their .NET applications. Full source code is included.

This document provides an overview of the package and links to documentation and examples.

2. Features

edtFTPnet/Free's many features include:

  • Rock-solid FTP transfers.
  • Passive and active modes are supported (PASV and PORT).
  • Resuming of interrupted binary transfers supported.
  • Includes a fully fledged Visual Studio component, FTPConnection.
  • Events for monitoring progress of data transfers.
  • Compatible with .NET 1.1 and .NET 2.0.
  • Compatible with Mono 1.2.x

3. Examples

Two examples are provided in the Examples directory. Both VB.NET and C# examples are provided.

CompleteFTP can be used as an FTP server on Windows. A trial can be downloaded to run the examples against. Even when the trial expires it can be used on localhost.

8. Other products

If you require secure file transfers or more advanced features, you may be interested in edtFTPnet/PRO, an extension of edtFTPnet/Free. It includes all of the free features, but in addition supports:

  • Secure FTP transfers via SFTP (FTP over SSH)
  • Secure FTP transfers via FTPS (FTP over SSL)
  • Seamless switching between FTP,SFTP and FTPS protocols
  • directory transfers
  • support for HTTP and SOCKS proxies
  • asynchronous methods

If a secure file transfer server is required, you may be interested in CompleteFTP. It is a Windows secure file server, and supports

  • Secure FTP transfers via SFTP (FTP over SSH)
  • Secure FTP transfers via FTPS (FTP over SSL)
  • HTTP and HTTPS
  • SCP - secure copy

It is also highly customizable, and supports enterprise features such as clustering.
