Getting Help

The Enterprise Distributed Technologies website offers a rich forum for discussion of topics related to edtFTPnet/Free.  It is fully searchable and contains advice and answers to many issues that developers have faced when using this library.

If you are trying to resolve a problem please search the forums to see if the issue has already been resolved. Also read How to diagnose problems.

If you are unable to find a solution, or suspect you have found a bug, you can post your problem or question on the forums.

For a faster response than the forums, contact us directly as outlined below.

How to contact support (for customers)

Customers (i.e. those who have purchased a source license) may e-mail support at support@enterprisedt.com. When emailing support with a problem, always include the following:
  • a stack trace (if an exception is being thrown) 
  • the section of the log file that illustrates the problem (see How to set up logging)
  • the type of server you are accessing (include the version if possible)
  • the version of edtFTPnet/Free you are using
    •
Please include as much information as possible; it will save us emailing you to ask for it.
