Product Overview



edtFTPnet/Free offers a set of robust and powerful, yet easy-to-use classes that make transferring files using the FTP protocol with the .NET Framework a breeze. 

edtFTPnet/Free is an extremely popular Open Source (LGPL) .NET FTP library that is widely tested and has an active user community.

edtFTPnet/Free's features include:

·     Rapid and Robust File Transfers via the FTP protocol
·     Active and Passive modes, Binary and ASCII transfers.
·     Transfer Resumption for interrupted file transfers.
·     FTP Streams for flexible reading from and writing to FTP servers.
